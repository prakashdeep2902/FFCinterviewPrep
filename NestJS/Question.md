If you're preparing for NestJS interviews, these are some of the **most frequently asked NestJS questions**, especially for Backend and Full Stack Developer roles:

### Beginner Level

1. What is NestJS and why is it used?
2. What are the main features of NestJS?
3. How is NestJS different from Express.js?
4. What is a Module in NestJS?
5. What is a Controller?
6. What is a Service/Provider?
7. Explain Dependency Injection in NestJS.
8. What are decorators in NestJS?
9. What is the purpose of `@Injectable()`?
10. What is the role of `main.ts`?

---

### Intermediate Level

11. Explain the NestJS request lifecycle.
12. What are Guards in NestJS?
13. Difference between Guards, Middleware, and Interceptors?
14. What are Pipes and when do you use them?
15. What is `ValidationPipe`?
16. How do you implement DTOs in NestJS?
17. What are Interceptors?
18. What are Exception Filters?
19. How do you handle global exception handling?
20. What is the difference between `useClass`, `useValue`, and `useFactory` providers?

---

### Authentication & Authorization

21. How do you implement JWT authentication in NestJS?
22. What is Passport.js and how does NestJS integrate with it?
23. Explain Auth Guards.
24. How do you implement Role-Based Access Control (RBAC)?
25. How do you secure API endpoints?

---

### Database Questions

26. How do you connect NestJS with PostgreSQL?
27. Difference between TypeORM and Prisma?
28. How do you define entities in TypeORM?
29. What are migrations?
30. How do you handle database transactions?
31. Explain Repository Pattern in NestJS.

### API Development

32. How do you create REST APIs in NestJS?
33. How do you implement API versioning?
34. How do you validate request data?
35. How do you implement pagination?
36. How do you upload files in NestJS?
37. How do you implement rate limiting?

### Advanced Level

38. What is Dynamic Module in NestJS?
39. Explain Custom Decorators.
40. How do you implement Caching?
41. What are Microservices in NestJS?

43. How do you implement WebSockets in NestJS?

### Testing

48. How do you write Unit Tests in NestJS?
49. What is the difference between Unit Testing and E2E Testing?
50. How do you mock dependencies in Jest?
51. How do you test Controllers and Services?

### Scenario-Based Questions (Very Common)

52. How would you design a scalable authentication system in NestJS?
53. How would you handle millions of API requests per day?
54. How would you implement refresh tokens securely?
55. How would you build a notification service using NestJS?
56. How would you implement caching for frequently accessed data?
57. How would you structure a large NestJS project?
58. How would you handle distributed transactions in microservices?
59. How would you secure sensitive environment variables?
60. How would you implement logging and monitoring in production?
