If you're preparing for JavaScript interviews, focus on the topics that repeatedly appear in frontend, full-stack, and Node.js interviews. Here's a prioritized roadmap:

## 1. JavaScript Fundamentals (Highest Priority)

- `var`, `let`, `const`
- Data types (primitive vs reference)
- Type coercion
- Equality (`==` vs `===`)
- Truthy and falsy values
- Scope (global, function, block)
- Hoisting
- Closures

**Common interview questions:**

- What is hoisting?
- Explain closures with a practical example.
- Difference between `let`, `const`, and `var`.

---

## 2. Functions

- Function declarations vs expressions
- Arrow functions
- Higher-order functions
- Callbacks
- Pure functions
- IIFE (Immediately Invoked Function Expression)

**Questions:**

- Difference between arrow functions and regular functions.
- What are higher-order functions?

---

## 3. Execution Context & Call Stack

- Global execution context
- Function execution context
- Call stack
- Memory creation phase
- Execution phase

**Questions:**

- How does JavaScript execute code?
- What happens when a function is called?

---

## 4. `this` Keyword (Very Common)

- Global context
- Object methods
- Arrow functions
- Explicit binding

Methods:

- `call()`
- `apply()`
- `bind()`

**Questions:**

- How does `this` work in JavaScript?
- Difference between call, apply, and bind.

---

## 5. Objects & Prototypes

- Object creation
- Prototype chain
- Inheritance
- Constructor functions
- ES6 classes

**Questions:**

- What is prototypal inheritance?

## 6. Arrays & Array Methods

Master these:

- `map()`
- `filter()`
- `reduce()`
- `find()`
- `some()`
- `every()`
- `forEach()`
- `sort()`

**Questions:**

- Implement map/filter/reduce manually.
- Difference between map and forEach.

## 7. Asynchronous JavaScript (Critical)

- Callbacks
- Promises

- What are promises?
- How does async/await work?

## Most Frequently Asked JavaScript Interview Topics (Top 10)

1. Closures
2. Hoisting
3. Event Loop
4. Promises & Async/Await
5. `this` keyword
6. Call, Apply, Bind
7. Prototypal Inheritance
8. Debouncing & Throttling

If you're targeting product companies (Google, Amazon, Microsoft, Meta, etc.), spend **60% of your time on Closures, Event Loop, Async JS, Prototypes, and Polyfills**, since these are the topics most likely to be tested deeply.
